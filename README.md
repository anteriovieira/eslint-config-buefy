# eslint-config-buefy

[![npm](https://img.shields.io/npm/v/eslint-config-buefy.svg)](https://www.npmjs.com/package/eslint-config-buefy)

Official ESLint rules for Buefy

## Usage

```bash
npm i -D eslint-config-buefy
```

Add it to your `.eslintrc.js` file:

```json
{
  "extends": [
    "buefy"
  ]
}
```
